# **App Name**: EquityMirror

## Core Features:

- Real-time Trade Mirroring: Replicates market, limit, stop, partial fills, cancels, and modifications from the master Alice Blue account to follower accounts with minimal delay.
- Configurable Risk Management: Enables setting lot multipliers, per-account caps, daily loss limits, and maximum exposure per symbol for each follower account.
- Detailed Logging and Intervention: Maintains comprehensive logs of all mirrored trades and any manual interventions made to manage risk.
- Automated Adjustment Tool: Generative AI adjusts limits of follower accounts and determines intervention threshold when markets shift rapidly based on sentiment and financial news inputs, within pre-set risk profiles as a tool.
- Configuration Management: Simple user interface with input validation for setting up the parameters, symbols and mapping Alice Blue user account to user accounts at the appropriate exchanges.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey reliability and trust in financial operations.
- Background color: Light gray (#ECEFF1), subtly desaturated, provides a clean and neutral backdrop for the application.
- Accent color: Yellow-amber (#FFAB40), analogous to blue, highlights critical actions and configurable parameters.
- Font pairing: 'Inter' (sans-serif) for both body text and headlines.
- A well-organized layout is very important, including clear navigation and well-defined panels for monitoring, setup, and reporting.
- Consistent, clear icons communicate complex trading concepts at a glance.
- Subtle animations when new order confirmations come through or thresholds are exceeded for easy acknowledgement of critical information.
